Краткое описание Sdk
--------------------
 
InetraAdv.1.0.1.arr включает в себя класс AdvPlayer. Этот класс является оберткой
вокруг стандартного Android MediaPlayer. В конструктор AdvPlayer передается фабрика,
которая создает объект Android MediaPlayer. Также передается объект 
android.view.ViewGroup - Layout на котором будет выводится как контент проигрываемый
плеером, так и реклама.

Реклама запускается перед каждым запуском (вызовом Play) плеера. 
У плеера есть свойство adsEnabled. true (по умолчанию) - рекламма будет запускаться
после кадого Play. Установка свойства в false отключает рекламу, проигрывается только контент.

Перед запуском рекламы срабатывает callback onAdStarted(), после окончания рекламного ролика или
если пользователь нажал кнопку "Пропустить" срабатывает onAdStopped(). Колбэки можно использовать
для скрытия/отображения контролов управления плеером (в состав Sdk не входят).

Пример использования плеера показан в прилагаемом тестовом приложении adv_player_example.
